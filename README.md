# Medical Report Summarizer

This project summarizes medical reports using a combination of BERT for feature extraction and Pegasus for text summarization. The user can input a medical report, and the system will return a summarized version of the report.

## Project Structure

- **data/**: Directory for raw and processed data.
- **models/**: Contains model code for BERT and Pegasus.
- **app/**: Flask web application code.
- **notebooks/**: Contains exploratory Jupyter notebooks.

## Running the Project

1. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```
